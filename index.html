How to use the tool

1. Go to the reservation page in Nova and select view Snapshots (bottom of the screen).

2. Select all text (CTR + A / CMD + A) and copy text.

3. Paste all text inside the textbox on the right and select the "View Snapshot" button.

4. Select an item in the left menu, of what you would like to see.
